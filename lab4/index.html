Lab 4- Pseudocoding and Problem Solving

Challenge

Create some functional pseudocode for an everyday task and a simple computer game

Problems

I had to look up the syntax for a javascript multiline comment, but that was about it.

Reflection

This assignment was a little out of the box for how I tend to approach problems, but it was interesting. Creating a simple computer game without a single line of code was a bit of a challenge, as I had to think in big picture steps rather than intricate details.

Pseudocode for an Everyday task: Making a Lasagna

  1. Buy Ingredients from Grocery Store
  2. Cook the Noodles
  3. Sautee Onions
  4. Cook Sausage
  5. Add Sauce
  6. Layer Cheese, Sauce, Meat
  7. Bake
  8. Enjoy

Pseudocode for a simple computer game

  1. Print new board
  2. Accept user input
  3. Validate user input
  4. Randomize computer response
  5. Check win condition or draw condition
  6. Add 1 point to winner or 0 for tie
  7. Repeat
